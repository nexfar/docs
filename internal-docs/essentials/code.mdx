## Código inline

Para denotar uma `palavra` ou `frase` como código, coloque-a entre crases (`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

## Blocos de código

Use [blocos de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) colocando o código entre três crases e siga as crases iniciais com a linguagem de programação do seu snippet para obter destaque de sintaxe. Opcionalmente, você também pode escrever o nome do seu código após a linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

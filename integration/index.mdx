## Objetivo

Esta seção documenta os métodos de integração disponíveis para que a Nexfar possa extrair dados do seu ERP. A integração permite que a plataforma Nexfar acesse informações essenciais como clientes, produtos, pedidos e condições comerciais diretamente do seu sistema.

## Escolhendo o Método Ideal

Cada método de integração possui características específicas:

### Banco de Dados:
- **Melhor para**: Acesso direto e em tempo real aos dados
- **Requisitos**: Liberação de IPs (whitelist) para acesso ao banco de dados via NAT ou via VPN Site-to-Site.
- **Vantagens**: Performance, dados atualizados, queries customizadas

### ERPs
- **Melhor para**: Integração diretas com ERPs: Winthor, Result, Target, InFarma, etc.
- **Requisitos**: Cada ERP tem requisitos individuais que devem ser observados
- **Vantagens**: Regras já mapeadas, integração já testadas em múltiplos clientes.

### REST API Nexfar
- **Melhor para**: Integrações modernas e flexíveis onde o distribuidor ou indústria nos envia os dados.
- **Requisitos**: Desenvolvimento de integração com a nossa API
- **Vantagens**: Flexibilidade, facilidade de manutenção e atualização

### FTP/EDI
- **Melhor para**: Transferência de arquivos EDI e/ou arquivos em lote
- **Requisitos**: Servidor FTP configurado
- **Vantagens**: Simplicidade, processamento batch, baixo acoplamento
